HTML

  • Elements are types of content on a webpage
  • Types of elements in HTML include text, image, header, list, or footer
  • Meta-related, content division, and semantic elements are also used in HTML
  • The head element contains info about the webpage
  • The head element contains metadata which is typically not visible to the user
  • The body element represents visible content for the user

CSS

  • There are 3 ways to use CSS to style a webpage
  • Frist, there is inline CSS, which styles directly in the HTML document
  • Second, you may use an internal CSS style sheet, contained in the head
  • Third, you may use an external CSS style sheet, which is the preferred method
  • External CSS style sheets keep all CSS rules in a separate file, which helps maintain separation of concerns
  • When using external CSS style sheets, you must link the CSS file in the head
  • The margin CSS rule indicates the space we want around the outside of an element
  • The padding CSS rule indicates the space we want around the content inside the element

Git

  • git status: checks which branch we are currently in
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commits all files in the working branch
  • git commit: commits changes in the repository
  • -m: associates a message with our commits
  • Example: git commit -m "added code to HTML file"
